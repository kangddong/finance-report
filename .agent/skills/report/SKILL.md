# Report

주식 정보를 분석하여 투자 조언을 포함한 자세한 보고서를 생성합니다.

## When to use this skill

- 주식 정보를 분석할 때
- 매수, 매도 타이밍을 추천받고 싶을 때
- 투자 조언을 받고 싶을 때
- 투자 조언에 대한 자세한 이유를 알고 싶을 때

## How to use it

- 종목정보는 `종목.md` 파일을 참고합니다.
- 리포트 작성 전 반드시 `종목.md`의 **보유 종목 전체 목록**과 **관심 종목 전체 목록**을 먼저 읽고 체크리스트로 적어둡니다.
- `종목.md`에 있는 항목은 임의로 생략하지 않습니다. 데이터가 부족한 경우에도 항목은 유지하고 `N/A`, `관망`, `추가 확인 필요`처럼 명시합니다.
- 보고서는 날짜별로 json 파일로 저장합니다. (예: `report/YYYY-MM-DD.json`)
- report/images/YYYY-MM-DD/에 캡쳐한 이미지가 있다면 분석에 참고하고, 보고서는 생성하지않습니다.
<!-- - 이미지가 없다면 toss-capture 스킬을 사용하여 이미지를 캡쳐합니다. -->

## 국장/미장 분리 구조

보고서는 `krSession`(국내장)과 `usSession`(미국장) 두 섹션으로 분리합니다.

- **국장 분석 시**: `종목.md`의 **국내 보유 종목**만 분석하여 `krSession.holdings`에 작성합니다.
- **미장 분석 시**: `종목.md`의 **해외 보유 종목**만 분석하여 `usSession.holdings`에 작성합니다.
- **관심 종목 작성 시**: `종목.md`의 **관심 종목 전체**를 기준으로 `watchlist`를 작성합니다.
- 기존 날짜 파일이 있으면 해당 섹션만 업데이트(덮어쓰기)합니다.

## Validation checklist

- `종목.md` 국내 보유 종목 수와 `krSession.holdings` 또는 최종 `holdings` 내 국내 종목 수가 일치하는지 확인합니다.
- `종목.md` 해외 보유 종목 수와 `usSession.holdings` 또는 최종 `holdings` 내 해외 종목 수가 일치하는지 확인합니다.
- `종목.md` 관심 종목 수와 최종 `watchlist` 수가 일치하는지 확인합니다.
- 불일치가 있으면 보고서를 저장하기 전에 누락 종목을 반드시 보완합니다.

### 국내 보유 종목 (krSession 대상)
- 삼성전자, SK하이닉스, PLUS K 방산, KODEX AI 전력핵심설비, 현대차, 두산에너빌리티

### 해외 보유 종목 (usSession 대상)
- 사이퍼 마이닝, 엔비디아, 팔란티어 테크

### 상세 분석 구조 (reason 필드)
보고서의 `reason` 필드에는 다음 내용을 포함하여 강동영님의 스타일에 맞춰 상세히 작성합니다:

1. **현재 주가 위치와 5일선 흐름**:
   - 단기 추세 (현재 주가가 5일선 위/아래인지)
   - 이동평균선 배열 상태 (정배열/역배열 등)
   - 이격도 과열 여부 (주가와 5일선의 거리)

2. **매수/매도 관점 분석 (5일선 기준)**:
   - **매수 관점**: 5일선 터치나 지지 시점 등 안전한 타점 제시
   - **보유 및 관망**: 추세 유지 시 보유 전략
   - **매도(주의) 시점**: 5일선 하향 돌파(이탈) 시 대응책
   - **손절 라인**: 손절 라인

3. **주요 가격 라인**:
   - 매도세가 예상되는 가격대:
   - 하락 시 지지가 예상되는 가격대: (5일선 위치 등)

4. **매매 동향 파악**:
   - 1달, 최근 1주일 매매 동향 파악:
   - 개인 매매 동향 파악
   - 기관 매매 동향 파악
   - 외국인 매매 동향 파악

5. **💡 총평**:
   - 한 줄 요약 및 사용자의 평단가 상황 등을 고려한 격려 메시지
